UPDATING AND MAINTENANCE OF DATABASES

  • You must monitor and optimize the database server to ensure database server availability and performance. Maintaining DB2 and Oracle databases involves tasks such as updating statistics, monitoring database, server, and space utilization, and planning backup and recovery strategies.

  • Applying the latest database client and server software fix pack on the application and database server

  • Always apply the latest fix pack to the database server and applications, as it contains troubleshooting and performance improvements. Both the database and the client must have the same fix pack level to avoid problems related to database version mismatches. Tuning the database to increase performanceo

  • You must ensure that the database is properly tuned. Databases are highly adjustable and you can modify them or configure their monitoring to increase performance. Tuning the database to improve response time can dramatically decrease the time users have to wait for operations to finish running, allows deployment to stay at the speed needed to run business, and optimizes resource usage to ensure existing hardware investments are used efficiently.
