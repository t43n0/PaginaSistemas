ACTUALIZACION Y MANTENIMIENTO DE BASES DE DATOS

  • Debe supervisar y optimizar el servidor de base de datos para asegurarse de la disponibilidad y el rendimiento de ésta. El mantenimiento de las bases de datos DB2 y Oracle implica tareas como actualizar estadísticas, supervisar la utilización de bases de datos, servidores y espacio, y planificar estrategias de copia de seguridad y recuperación.

  • Aplicación del último fixpack del software del servidor y el cliente de base de datos en el servidor de aplicaciones y de base de datos

  • Aplique siempre el último fixpack en el servidor de base de datos y aplicaciones, ya que contiene soluciones para problemas y mejoras para el rendimiento. Tanto la base de datos como el cliente han de tener el mismo nivel de fixpack para evitar problemas relacionados con la discrepancia de versiones de la base de datos. Ajuste de la base de datos para aumentar el rendimiento

  • Debe asegurarse de que la base de datos esté correctamente ajustada. Las bases de datos son altamente ajustables y puede modificarlas o configurar su supervisión para incrementar el rendimiento. Ajustar la base de datos para mejorar el tiempo de respuesta puede disminuir radicalmente el tiempo que los usuarios tienen que esperar para que las operaciones terminen de ejecutarse, permite que la implementación se mantenga a la velocidad necesaria para dirigir los negocios y optimiza el uso de los recursos para garantizar que se utilicen con eficiencia las inversiones de hardware existentes.
